## ðŸš€ Installation

You can add these sections to your Shopify store without installing any third-party apps.

### Step 1: Download the Code
1. Navigate to the file you want to use in this repository (e.g., `sections/custom-slider.liquid`).
2. Copy the entire code.

### Step 2: Add to Shopify Theme
1. Go to your Shopify Admin Dashboard.
2. Navigate to **Online Store > Themes**.
3. Click the **... (three dots)** button next to your active theme and select **Edit code**.
4. Scroll down to the **Sections** folder and click **Add a new section**.
5. Name the file exactly as instructed in the specific file's comments (or give it a relevant name like `custom-slider`).
6. Paste the copied code into the file and hit **Save**.

### Step 3: Activate in Customizer
1. Go back to **Online Store > Themes** and click **Customize**.
2. Navigate to the page where you want to add the section.
3. Click **+ Add Section** on the left sidebar.
4. Search for the section name (e.g., "Custom Slider") and select it.
5. Configure the settings (images, text, colors) using the sidebar options.
